# OneDmin.jl

## How to install

Install and test the OneDmin package with the Julia package manager:
```julia
pkg> add OneDmin
pkg> test OneDmin
```
You can access the most up-to-date version of the OneDmin package using:
```julia
pkg> add https://github.com/vepiteski/OneDmin.jl
pkg> test OneDmin
```
